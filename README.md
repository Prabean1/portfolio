## ‚ö° About The Project
I built this site to escape the "sanitized" feel of drag-and-drop builders like Wix or Squarespace. I wanted granular control over the logic, the layout, and the performance.

This repository serves as the codebase for my personal portfolio, featuring a custom-built project loader and a dynamic timeline system.

### Key Features
*   **Dynamic Content Engine:** Projects and Timeline entries are stored in structured JSON objects and rendered via JavaScript, separating data from presentation.
*   **Gamified Timeline:** A vertical timeline system that tracks engineering and leadership milestones, complete with interactive "Side Quest" elements.
*   **Custom Visuals:** Integrated **Three.js** for WebGL grid distortion effects and background interactions.
*   **Responsive Architecture:** Built on **Bootstrap 5** but heavily customized for a specific dark-mode aesthetic.
